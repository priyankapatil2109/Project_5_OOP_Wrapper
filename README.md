# Project_5_OOP_Wrapper
# Employee Management System (OOP Wrapper Project)

## Project Overview

The Employee Management System is a Python-based console application developed using Object-Oriented Programming (OOP) concepts. This project manages employee information and demonstrates important OOP principles such as Encapsulation, Inheritance, Method Overriding, Constructor, Destructor, Getter and Setter Methods, Method Overloading, super(), and issubclass().

---

## Objective

The objective of this project is to build an Employee Management System that:

- Stores employee details.
- Manages managers and developers as specialized employees.
- Demonstrates OOP concepts in Python.
- Provides a menu-driven user interface for user interaction.

---

## Features

### Employee Class
- Employee ID
- Name
- Age
- Salary
- Getter and Setter Methods
- Constructor and Destructor

### Manager Class
- Inherits from Employee
- Additional Department Attribute
- Overridden display() Method

### Developer Class
- Inherits from Employee
- Additional Programming Language Attribute
- Overridden display() Method

### OOP Concepts Implemented
- Classes and Objects
- Encapsulation
- Inheritance
- Method Overriding
- Constructor
- Destructor
- Getter and Setter Methods
- super()
- issubclass()
